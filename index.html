INTERNAL SCREENER · SG Score v1.4
SG Score v1.4 — formula & components
SG Score  =  C1×0.28 + C2×0.22 + C3×0.15 + C4×0.13 + C5×0.12 + C6×0.07 + C7×0.03
C1 · 28% Risk-Adjusted Momentum
20d return / ATR%, percentile-binned across the live universe. Capped to ±5.0 to suppress low-vol outliers.
C2 · 22% Relative Strength vs SPY
Stock 20d return − SPY 20d return. Bins at +5/+2/-2/-5%.
C3 · 15% Volume Confirmation
Up-day vol / down-day vol over last 10 sessions.
C4 · 13% Sector Context
Sector ETF 20d return vs SPY. Currently PARTIAL — breadth + F&G data not wired (C4 capped at 9).
C5 · 12% Trend Quality
EMA stack 10/20/50, slope alignment, distance from 20 EMA, 50 vs 200 SMA. −2 if extended >15% above 20 EMA.
C6 · 7% Options Flow / Sentiment
Currently NEUTRAL 5 — call volume / P/C ratio data not yet fetched. Flagged DATA LIMITED.
C7 · 3% Execution Quality
OI + spread on the target option. NEUTRAL 5 + PROVISIONAL when chain not fetched. Run OVTLYR Fetch Prices to populate spreads.
Hard gates (DQ unless soft mode)
earnings <7d · 30d avg vol <1M · price <$20 · OI <250 · spread >10% · price < 10 EMA
Recommendations
STRONG BUY ≥ 8.5   ·   BUY ≥ 7.8   ·   WATCH ≥ 7.0   ·   SKIP < 7 or any DQ
